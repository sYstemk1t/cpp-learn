# 分支与循环

## 程序结构化

结构化程序设计思想是：把一个需要解决复杂问题分解为若干个模块来处理，每个模块解决一个小问题，这种分而治之的方法大大降低了程序设计的难度，结构化程序设计的核心问题是算法和控制结构；

代码详见：ProgramStructure
